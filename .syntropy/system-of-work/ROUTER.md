# Router (Repo Collaboration Domains)

Use this file when you don’t know which domain owns a change.

## How to Route Work

1. Read `.syntropy/system-of-work/EXECUTION_CONTRACT.md`
2. Pick the primary domain below
3. Open that domain’s `AGENT.md` and follow its rules/workflows
